# Support

Thank you for using **San Orien Symbols**.

## Before asking for help

Please check:

* Existing Issues
* Existing Discussions
* The README
* The documentation

Your question may already have been answered.

---

## Bug Reports

If you found a bug, please create a **Bug Report** issue using the provided template.

---

## Feature Requests

Have an idea?

Create a **Feature Request** issue.

---

## Icon Requests

Need a new icon?

Use the **Icon Request** template and explain your use case.

---

## External Usage Requests

Using San Orien Symbols outside the San Orien ecosystem requires approval.

Please use the **External Usage Request** issue template.

---

## Discussions

General questions, ideas, and feedback belong in GitHub Discussions.

---

## Response Times

This project is maintained in my spare time.

While I try to respond as quickly as possible, response times are not guaranteed.
