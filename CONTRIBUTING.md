# Contributing to San Orien Symbols

Thank you for your interest in contributing!

## Before You Start

Please search existing Issues before creating a new one.

## Bug Reports

Use the **Bug Report** template.

## Feature Requests

Use the **Feature Request** template.

## Icon Requests

Use the **Icon Request** template.

## External Usage

If you wish to use San Orien Symbols outside the San Orien ecosystem, please use the **External Usage Request** template.

Approval is handled individually and is entirely at the discretion of the project owner.

## Pull Requests

Please:

* keep changes focused;
* follow the existing design language;
* provide clean SVGs;
* avoid unnecessary formatting changes;
* explain your changes clearly.

Submitting a Pull Request does not guarantee that it will be merged.

By submitting a Pull Request, you agree that your contribution may be incorporated into San Orien Symbols and distributed under the San Orien Symbols License.
